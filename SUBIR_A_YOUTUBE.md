# Videos para Subir a YouTube

## Instrucciones
1. Abre https://studio.youtube.com
2. Click en **CREAR** → **Subir videos**
3. Arrastra los videos o selecciónalos
4. Copia el título y descripción de cada video de abajo
5. Configura como **No listado** (solo accesible con enlace)
6. **Guarda el enlace de cada video** para actualizar el sitio web
